- Duration long: 400ms

- Easing spring: cubic-bezier(0.34, 1.56, 0.64, 1)

- Reduced-motion: prefers-reduced-motion respected globally

## Anti-patterns (forbidden)
- No pure white (#fff) on dark backgrounds — use #f0fdf6 max
- No hard box-shadow on dark surfaces — use glow or subtle drop
- No excessive blur (>20px) — keep backdrop-blur at 12-16px
- No inconsistent border-radius mixing
- No animation duration > 500ms (feels sluggish)
- No unstyled focus states — always use ring pattern

## Stack-specific rules
- React 19 with Tailwind CSS v4 @theme directive
- No shadcn/ui — custom components using Tailwind utilities
- Use CSS custom properties via @theme for design tokens
- Prefer `@apply` sparingly — inline Tailwind classes preferred
- GPU-accelerated animations (transform, opacity) only
- All interactive elements need focus-visible ring
