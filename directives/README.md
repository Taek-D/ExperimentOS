# Directives (지시사항)

이 디렉토리에는 **Layer 1: Directive** 파일들이 포함됩니다.

## 목적

디렉티브는 마크다운 형식으로 작성된 SOP(Standard Operating Procedures)입니다. 각 디렉티브는 특정 작업에 대한 자연어 지침을 제공합니다.

## 디렉티브 작성 가이드

각 디렉티브 파일에는 다음 내용이 포함되어야 합니다:

1. **목표 (Goal)**: 이 작업이 달성하고자 하는 것
2. **입력 (Inputs)**: 필요한 데이터 또는 파라미터
3. **도구/스크립트 (Tools/Scripts)**: 사용할 `execution/` 디렉토리의 스크립트
4. **출력 (Outputs)**: 예상되는 결과물 (Google Sheets, Slides 등)
5. **엣지 케이스 (Edge Cases)**: 주의해야 할 특수 상황

## 예시 구조

```markdown
# [작업명] 디렉티브

## 목표
이 작업의 목적을 설명합니다.

## 입력
- 필요한 데이터 1
- 필요한 데이터 2

## 사용 도구
- `execution/script_name.py` - 스크립트 설명

## 출력
- Google Sheets: [시트 이름]
- 기타 산출물

## 엣지 케이스
- 주의사항 1
- 주의사항 2

## 학습된 내용
(작업을 수행하면서 발견한 팁, API 제한사항, 타이밍 이슈 등)
```

## 살아있는 문서

디렉티브는 **살아있는 문서**입니다. 작업을 수행하면서 새로운 것을 배울 때마다 해당 디렉티브를 업데이트하세요.
